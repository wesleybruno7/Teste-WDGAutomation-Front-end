Sobre o teste

Linguagens utilizadas

HTML5

Primeiramente utilizei o HTML 5 para criar as estruturas/conteúdos básicos.

CSS3

Com CSS3 criei um custom.css para definir estilos específicos que não consegui apenas utilizando Bootstrap 4.

Javascript

Para executar ações ou interações utilizei o Javascript como mostrar/ocultar conteúdos, preencher campos, etc.

Conceitos, frameworks e outros

Arquitetura MVC (Model-View-Controller)

Neste projeto tentei aplicar a Arquitetura MVC para separar e organizar melhor o código (criando VIEWs para o conteúdo em HTML/CSS/Angular, CONTROLLERs para funções e regras de negócio em Javascript/Angular, apenas a MODELs não utilizei, pois de imediato não tínhamos a necessidade de comunicar com um banco de dados, as chamadas de API ficaram nas CONTROLLERs mesmo).

Bootstrap 4

Utilizei o Bootstrap 4 para estilizar quase todos os conteúdos/páginas, utilizei também o esquema de modal do proprio Bootstrap quando altera/exclui um usuário.

Angular.js

Não conhecia este framework até o início deste teste, acabei estudando-o no final de semana e consegui aplicar algumas dinâmicas e esquemas para single page como solicitado. Com ele fiz o esquema de rotas, criei algumas funções/services. No projeto acabei utilizando o JSON Web Token (JWT) em conjunto para o esquema de login para evitar que um usuário sem credencial acesse os conteúdos privados da Single-Page Application(SPA).

Jquery

A principio estava utilizando o Jquery para utilizar o ajax, mas depois acabei substituindo pelo $http.get (por exemplo) do Angular.js. Ficou no projeto apenas devido a necessidade que o Bootstrap utiliza-se do jquery para realizar algumas ações.

Sites responsivos

Tentei deixar os conteudos responsivos (principal motivo de utilizar o Bootstrap 4, pois ele facilita a criação e estilização de sites responsivos).
